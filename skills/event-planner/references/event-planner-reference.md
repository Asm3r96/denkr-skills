# Event Planner Reference

- Prefer one events notebook with `writingRules` like one page per event.
- Useful page sections:
  - Overview
  - Date and place
  - Guests
  - Tasks
  - Budget
  - Supplies
  - Notes
  - Follow-ups
- Good markdown structures:
  - bullet lists for guest names or supplies
  - numbered lists for prep sequences
  - checkbox lists for event tasks like `- [ ] Buy candles`
- Use reminders only for meaningful future dates such as invitations, booking deadlines, shopping days, or the event day.
