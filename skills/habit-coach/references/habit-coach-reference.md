# Habit Coach Reference

- Prefer one habits notebook with practical `writingRules`, unless the user clearly wants one notebook per habit.
- Good `writingRules` examples:
  - one overview page per habit and one monthly check-in page
  - one page per habit with dated sections appended over time
- Useful page sections:
  - Goal
  - Why this matters
  - Cadence
  - Rules
  - Obstacles
  - Wins
  - Next review
- Good markdown structures:
  - bullet lists for habits rules
  - numbered lists for routines
  - checkbox lists like `- [ ] Evening walk`
- Use reminders only when the user wants future nudges or the habit clearly needs scheduled support.
