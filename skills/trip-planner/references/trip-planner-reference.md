# Trip Planner Reference

- Prefer one trips notebook with `writingRules` like one page per trip.
- Useful page sections:
  - Overview
  - Dates
  - Bookings
  - Places
  - Packing
  - Budget
  - Notes
- Good markdown structures:
  - bullet lists for places or ideas
  - checkbox lists for packing and prep
  - short headings for stable trip sections
- Use `notebook.read` with `action: list_pages` and a trip-name query before creating a new trip page.
- Use reminders only for meaningful dates such as booking deadlines, packing day, airport departure, or return travel.
