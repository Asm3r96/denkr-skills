# Idea Incubator Reference

- Prefer one ideas notebook with `writingRules` like one page per idea.
- Useful page sections:
  - Concept
  - Why it matters
  - Questions
  - Assumptions
  - Risks
  - Next steps
  - Experiments
- Good markdown structures:
  - bullet lists for open questions
  - numbered lists for development sequences
  - checkbox lists for next-step tasks
- Use reminders only when the user wants a revisit, deadline, or follow-up prompt.
