# Decision Helper Reference

- Prefer one decisions notebook with `writingRules` like one page per decision.
- Useful page sections:
  - Decision
  - Options
  - Criteria
  - Pros
  - Cons
  - Unknowns
  - Recommendation
  - Next step
- Good markdown structures:
  - bullet lists for pros and cons
  - numbered lists for ordered options
  - checkbox lists for follow-up actions after the decision
- Use reminders only when the user wants a later review or follow-up action.
